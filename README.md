# DataStructures

Đây là repo mẫu chứa ví dụ về nguyên tắc *composition over inheritance* trong một ứng dụng console C#.

## Chạy thử

```
dotnet run
```

Lệnh sẽ biên dịch và thực thi `Program.cs`, minh họa cách thay đổi hành vi chiến đấu của `Character` bằng cách hoán đổi các lớp `IAttackBehavior`.
